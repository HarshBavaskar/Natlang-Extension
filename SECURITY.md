# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do **NOT** open a public issue for security-related items. Instead, report security vulnerabilities privately to [harsh.bavaskar@example.com](mailto:harsh.bavaskar@example.com).

When reporting a vulnerability, please provide:
- A clear, concise description of the issue.
- Step-by-step instructions to reproduce the vulnerability (including target provider and payload).
- Any potential impact and possible mitigations.

You should receive a response within 48 hours.
